About

bio.dat

I'm a hands-on automation and maintenance technician with over a year of industrial experience in Japanese and multinational manufacturing environments. My work centers on control panel assembly, motor control, instrumentation, and troubleshooting — the kind of work that rewards patience, staged testing, and a healthy respect for grounding checks.

I trained in a culture of kaizen and preventive maintenance rather than firefighting: verify before you power on, document what you find, and fix the root cause instead of the symptom. That discipline now carries over into the embedded and IoT projects I build outside of work — from PLC-driven SCADA systems to computer-vision sorting rigs.

Apple Ripeness Sorting Conveyor

Vision / AI

A conveyor-based sorting rig that classifies apples by ripeness in real time. Runs inference on a Hailo AI accelerator paired with a Raspberry Pi 5, with hyperparameter optimization used to tune the classification model for accuracy and inference speed on constrained hardware.

Raspberry Pi 5 Hailo AI Computer Vision Python

SCADA Control System — 3-Phase AC Motor

PLC / SCADA

A supervisory control system for a three-phase AC motor built on a Schneider TM221 PLC driving an ATV320 variable-frequency drive. Covers ladder logic for motor start/stop/speed sequencing plus a SCADA layer for real-time monitoring and control.

Schneider TM221 ATV320 VFD SCADA Ladder Logic

Automated Enema Medicine Stirring System

Embedded

An ESP32-driven stirring mechanism for enema medicine preparation, controlled via IR remote for hands-free, contactless operation in a clinical-adjacent setting — prioritizing repeatable timing and simple, reliable control over a minimal interface.

ESP32 IR Remote Motor Control

ESP32 Briquette Drying System

Embedded

A closed-loop drying system for biomass briquettes, using a Type-K thermocouple for high-temperature sensing and an ESP32 to regulate the drying cycle against a target temperature profile.

ESP32 Type-K Thermocouple Closed-loop Control
